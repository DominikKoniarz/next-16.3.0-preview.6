# Next.js 16.3 preview sandbox

Exploring Next.js `16.3.0-preview.6` App Router patterns:

-   **Cache Components** (`cacheComponents`) + `'use cache'` / tags
-   **Partial prefetching**
-   **React Compiler**
-   **Feature-sliced layout** (`features/`) — pages compose, features own queries/UI; guided by [nextjs-app-architecture](.agents/skills/nextjs-app-architecture/SKILL.md) (aurorascharff)
-   **Custom Locale routing** via `proxy.ts` (`en` / `pl`)
-   RSC defaults: async server components, Suspense at the page, `params.then()`
